# Моделирование устойчивости колебаний расточной борштанги

## Описание проекта

Программный комплекс предназначен для численного анализа устойчивости крутильных, продольных и поперечных колебаний расточной борштанги при обработке глубоких отверстий.

Проект реализует математические модели динамической системы резания с учетом:

- внутреннего трения материала борштанги,
- динамической жесткости процесса резания,
- регенеративного эффекта,
- запаздывания стружкообразования,
- геометрических параметров инструмента.

Работа выполнена в рамках курсового проекта по направлению «Фундаментальная информатика и информационные технологии».

## Теоретическая основа

### Крутильные колебания

Модель основана на уравнении крутильных колебаний стержня с учетом внутреннего трения:

GJ ∂²φ/∂x² + GJ δ₁ ∂³φ/∂x²∂t − ρJ ∂²φ/∂t² = 0

После перехода в частотную область (p = iω) используется функция D-разбиения:

σ(p) = −p − λ₁ √(1 + δ₁ p) coth(λ₂ p / √(1 + δ₁ p))

где  
λ₁ = √(ρG) Jp / Jr  
λ₂ = L √(ρ/G)

Граница устойчивости определяется условием:

Im(σ(iω)) = 0

---

### Продольные колебания

Учитывается влияние регенеративного запаздывания τ:

ES ∂²w/∂x² + ES δ₃ ∂³w/∂x²∂t − ρS ∂²w/∂t² = F(τ)

Строится параметрическая кривая:

K₁(ω) – δ(ω)

Контрольные значения при ω → 0:

K₁₀ = ES / (L (1 − μ))  
δ₀ = − ES μ τ / (L (1 − μ))

---

### Поперечные колебания

Используется модальная аппроксимация балки с учетом:

- изгибной жесткости,
- динамической жесткости резания,
- регенеративной составляющей,
- фазового сдвига.

Строится годограф комплексной функции W(p).  
Граница устойчивости определяется условием:

Im(W(iω)) = 0

---

## Реализованные методы анализа

В программе реализованы:

- метод D-разбиения,
- построение годографов,
- поиск пересечений Im = 0,
- вычисление критической частоты ω*,
- расчет частоты f* = ω*/(2π),
- параметрический анализ влияния δ₁, μ, τ, L,
- использование типовых пресетов параметров.

## Структура проекта

### Основные модули

- `main.py` — точка входа в приложение
- `main_window.py` — главное окно
- `menu_page.py` — меню навигации
- `torsional_page.py` — анализ крутильных колебаний
- `longitudinal_page.py` — анализ продольных колебаний
- `transverse_page.py` — анализ поперечных колебаний
- `algorithm_page.py` — описание алгоритма работы
- `help_page.py` — справка
- `borebar_model.py` — математические модели
- `borebar_gui.py` — расширенный графический интерфейс

### Тестирование

Реализованы регрессионные тесты:

- `test_torsional_regression.py`
- `test_longitudinal_regression.py`
- `test_transverse_regression.py`

Проверяется:

- соответствие аналитическим формулам,
- корректность предельных значений,
- отсутствие бесконечных значений,
- корректность пересечения Im = 0.
