# Исследование крутильных и продольных колебаний расточной борштанги

## О проекте

Данный проект представляет собой курсовую работу, выполненную в рамках направления «Фундаментальная информатика и информационные технологии».  
Цель работы — исследование устойчивости состояния равновесия системы, описывающей крутильные и продольные колебания расточной борштанги при обработке глубоких отверстий.

Разработана математическая модель колебаний, реализована программа на Python с графическим интерфейсом для визуализации результатов метода D-разбиения, проведены численные эксперименты.

---

## Математическая модель

### Крутильные колебания
Уравнение крутильных колебаний борштанги с учётом внутреннего трения:

GJρ·φ''(x,t) + GJρ·δ₁·φ̇''(x,t) - Jρ·ρ·φ̈(x,t) = 0

Краевые условия:
- Жёсткое закрепление в суппорте: \(\varphi(0,t) = 0\)
- На конце с расточной головкой учитываются моменты сил резания и трения

### Продольные колебания
Уравнение продольных колебаний с учётом сил запаздывания:

ES·w''(x,t) + ES·δ₃·ẇ''(x,t) - ρ·S·ẅ(x,t) = 0

Краевые условия включают силы трения и упругости на конце с головкой.

---

## Метод исследования

Для анализа устойчивости используется **метод D-разбиения**:
- Для крутильных колебаний — по одному комплексному параметру \(\widehat{\delta}\)
- Для продольных колебаний — по двум действительным параметрам \(K_1\) и \(\delta\)

## Программная реализация

Проект включает следующие файлы:

- `main.py` — запуск приложения
- `borebar_gui.py` — графический интерфейс (PyQt5)
- `borebar_model.py` — математическая модель и расчёты
- `Отчет_по_практике_Дудченко_Олеся_2025_ФИИТ_3_курс.docx` — полный текст отчёта

### Функционал программы:
- Ввод параметров борштанги и режимов обработки
- Построение кривых D-разбиения для крутильных и продольных колебаний
- Интерактивное изменение параметров (длина, трение, запаздывание)
- Экспорт результатов в JSON или CSV
- Визуализация в реальном времени

## Результаты исследований

### Крутильные колебания:
- С увеличением длины борштанги область устойчивости сужается
- Увеличение коэффициента трения расширяет область устойчивости

### Продольные колебания:
- Увеличение времени запаздывания и длины борштанги снижает устойчивость
- Рост коэффициента трения улучшает устойчивость системы
