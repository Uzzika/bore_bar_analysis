# Моделирование колебаний расточной борштанги

## О проекте

Это приложение предназначено для численного анализа устойчивости расточной борштанги при обработке глубоких отверстий. В программе рассматриваются три типа колебаний:

- крутильные;
- продольные;
- поперечные.

Для каждого типа колебаний пользователь может задать параметры системы, построить расчетную кривую, найти характерные точки потери устойчивости и сохранить результаты в файл.

Проект выполнен как учебно-исследовательская работа и представляет собой рабочий программный комплекс для сравнительного анализа моделей, визуализации результатов и проверки расчетов.

---

## Что умеет программа

Приложение позволяет:

- рассчитывать крутильные, продольные и поперечные колебания борштанги;
- строить кривые D-разбиения и годографы;
- находить точки, в которых мнимая часть расчетной функции обращается в ноль;
- определять критическую частоту и соответствующую ей граничную точку устойчивости;
- использовать готовые пресеты параметров;
- экспортировать результаты расчета в JSON и CSV;
- работать через графический интерфейс на PyQt5.

---

## Реализованные модели

### Крутильные колебания

Крутильная модель основана на уравнении колебаний упругого стержня с учетом внутреннего трения материала. В программе строится кривая D-разбиения `σ(iω)` и анализируются точки пересечения условия:

`Im(σ(iω)) = 0`

По этим точкам определяется критическое значение, которое используется для оценки границы устойчивости.

В расчетной части используется физическая положительная ветвь частот, а при необходимости симметричное отображение графика строится как зеркальное представление для визуализации.

### Продольные колебания

Продольная модель реализована в физически согласованной SI-постановке. В программе рассчитывается параметрическая кривая:

`K₁(ω) – δ(ω)`

где учитываются жесткость системы, плотность материала, длина борштанги, регенеративная связь и время запаздывания.

Для продольной ветки программа также определяет характерные точки на кривой и отображает контрольные параметры модели.

### Поперечные колебания

Поперечная модель реализована в одномодовой постановке методом Бубнова–Галёркина. В программе строится годограф комплексной функции `W(iω)` и анализируются точки, где выполняется условие:

`Im(W(iω)) = 0`

При расчете учитываются геометрия борштанги, изгибная жесткость, динамическая жесткость резания, внутреннее трение и параметры регенеративной связи.

На текущем этапе проект использует рабочую одномодовую реализацию, пригодную для численного анализа и воспроизведения расчетных зависимостей, применяемых в приложении.

## Интерфейс программы

В приложении доступны следующие разделы:

- **Главное меню** — переход к разделам анализа;
- **Крутильные колебания** — ввод параметров, построение кривой `σ(iω)`, поиск критических точек;
- **Продольные колебания** — ввод параметров, построение кривой `K₁(ω)–δ(ω)`;
- **Поперечные колебания** — ввод параметров, построение годографа `W(iω)` и анализ устойчивости;
- **Справка** — краткое описание принятых моделей;
- **Алгоритм работы** — инструкция по использованию программы.

---

## Основные файлы проекта

- `main.py` — точка входа в приложение;
- `main_window.py` — главное окно и навигация между страницами;
- `menu_page.py` — стартовое меню;
- `torsional_page.py` — страница анализа крутильных колебаний;
- `longitudinal_page.py` — страница анализа продольных колебаний;
- `transverse_page.py` — страница анализа поперечных колебаний;
- `help_page.py` — страница справки;
- `algorithm_page.py` — страница с инструкцией по работе;
- `borebar_model.py` — математические модели и вычислительная логика;
- `presets.py` — набор типовых параметров для разных режимов;
- `export_utils.py` — экспорт результатов;
- `summary_builders.py` — формирование текстовых сводок по расчетам;
- `borebar_gui.py` — совместимый файл запуска для старого имени GUI-модуля.

## Экспорт результатов

Результаты расчета можно сохранять в форматах **JSON** и **CSV**.

Экспорт включает:

- тип анализа;
- использованные параметры;
- сведения о модели;
- рассчитанную кривую;
- найденные специальные точки;
- критическую точку устойчивости;
- краткую численную информацию о расчете.

Это позволяет использовать программу не только для визуального анализа, но и для последующей обработки результатов.
